# Agentic Registration Prompt

在官方 Agentic Wallet / OnchainOS 注册界面中，使用官方提示词：

`Register me for the Agentic Trading Contest`

## Safety Reminders

- 仅使用一个已注册的 Agentic Wallet 账号参与比赛。
- 如果钱包已经导出过，停止注册或交易流程，先回到官方规则核对资格。
- 不要在提示词、聊天记录、日志或本地文件中粘贴邮箱、验证码、私钥、助记词或钱包地址。
- 仅在官方 Agentic Wallet / OnchainOS 环境内手动完成注册确认，不在本仓库脚本中接入真实钱包。
